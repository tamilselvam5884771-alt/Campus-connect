# CampusConnect - A Smart Platform to Improve Campus Life

**Build with MERN 🍃🔥⚛️ Node**

CampusConnect is a smart college issue and resource platform that bridges the gap between students and administration.

## Core Features
1.  **Student Report Issues & Request Resources**: Easily report infrastructure problems or request necessary equipment.
2.  **Admin / Faculty Dashboard**: Efficiently manage and track reports.
3.  **Issue Status Tracking**: Real-time updates on reported issues.
4.  **Role-Based Authentication**: Secure access for Students, Faculty, and Admins.
5.  **Upvote & Priority System**: Highlight the most critical issues.

## Tech Stack
-   **Client**: React.js, Tailwind CSS, Vite
-   **Server**: Node.js, Express.js
-   **Database**: MongoDB

## Getting Started

### Prerequisites
-   Node.js installed
-   MongoDB installed or MongoDB Atlas URI

### Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd CampusConnect
    ```

2.  **Setup Server**
    ```bash
    cd server
    npm install
    # Create a .env file with your MONGO_URI
    npm start
    ```

3.  **Setup Client**
    ```bash
    cd client
    npm install
    npm run dev
    ```
